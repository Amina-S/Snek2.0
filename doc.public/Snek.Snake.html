Module Snek.Snake

module Snake: SnakeSig 
type t

Returns the list of points occupied by s. The last element of the list is the 'head' of the snake, and the first element of the list is the 'tail' of the snake (that is about to disappear if the move is allowed).

val body : t -> Point.t list

Returns an object of type dir that corresponds to the current direction of s.

val direction : t -> Snek.dir

Takes in a Point.t list object and a dir object. Initializes s of type Snake.t with the given inputs.

val init : Point.t list -> Snek.dir -> t
